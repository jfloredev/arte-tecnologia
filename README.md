# Memorias Conectadas

Aplicación web interactiva para compartir y visualizar memorias colectivas.

## Instalación

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm start
```

La aplicación se abrirá en [http://localhost:3000](http://localhost:3000)

## Características

- Panel de introducción con información del proyecto
- Panel para compartir memorias con selección de emociones
- Panel de memoria colectiva con visualización de todas las memorias
- Interfaz responsive y minimalista
- Modal para ver detalles de cada memoria

## Tecnologías

- React 18
- CSS3
- JavaScript ES6+
